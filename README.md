This project demonstrates a complete end-to-end data analysis workflow,starting from data extraction to generating business insights. 
The dataset was programmatically fetched using the Kaggle API, processed and cleaned using Python in Jupyter Notebook,
and then loaded into MySQL for advanced SQL-based analysis.

The goal of the project is to analyze retail sales data and extract meaningful insights related to product performance,
regional trends, and business growth.

Project Workflow:
    1. Data Extraction (Kaggle API)
    2. Data Cleaning & Processing (Python - Jupyter Notebook)
    3. Data Storage (MySQL Workbench)
    4. Data Analysis (SQL)

Tools & Technologies Used:
    * Python (Pandas, Jupyter Notebook)
    * Kaggle API
    * MySQL (MySQL Workbench)
    * SQL (CTEs, Window Functions, Aggregations)

Key Learnings:
    * Built a complete data pipeline from data extraction to analysis.
    * Gained hands-on experience with real-world datasets.
    * Strengthened SQL skills (CTEs, window functions, aggregations)
    * Learned integration between Python and MySQL.
